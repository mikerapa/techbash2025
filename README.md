# TechBash 2025
Thank you for being part of TechBash 2025. In this repository, you'll find session decks, sample code, links, and other useful information from the presenters. Be sure to use #TechBash when posting in social media.

### Speakers
Please feel free to send pull requests with any of your content that you want to share with the TechBash attendees. This will help us host a single place for attendees to get the content from the conference. Drop your session slides, sample code and demos, links to online content, and anything else that's useful. If there are other locations on GitHub, please add a Markdown file in your session folder with a link to that repo. Your session folder should be the title of your session. Send a pull request and we will merge it in.

### Sponsors
Fork this repository and put links or anything else in a subdirectory named with the same name of your organization. Then send a pull request and we will merge it in.
